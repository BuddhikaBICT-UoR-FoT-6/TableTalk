# Changelog

All notable changes to TableTalk will be documented in this file.

## [1.0.0] - 2025-06-15
### Added
- Complete business analysis documentation (Use cases, ERD, User Stories, Test Cases).
- MIT License and README.
- Light and Dark mode toggle.
- Real-time chat system between chefs and customers.
- HTML email receipts for customers.
- Comprehensive Sri Lankan cuisine menu.
- Secure image uploading for new dishes.

### Fixed
- Addressed 404 image errors by ensuring correct filesystem naming conventions and routing.
- Fixed 400 Bad Request error on feedback submission.
- Resolved "ghost message" bug for unassigned tables.

### Changed
- Replaced native browser `alert()` dialogs with custom Toast Notifications.
- Implemented loading spinners on critical actions (Login, Place Order, Add Dish) for better UX.
