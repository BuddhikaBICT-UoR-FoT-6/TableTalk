# TableTalk

TableTalk is a modern, responsive, and seamless restaurant ordering and management system. It allows customers to place orders directly from their tables, chefs to manage the kitchen queue in real-time, and admins to oversee feedback and operations.

## Features
- **Customer Portal**: Browse a rich Sri Lankan menu, manage cart, place orders, and submit feedback.
- **Kitchen Dashboard**: Real-time order management and direct chat with tables.
- **Admin Dashboard**: Analytics, feedback monitoring, and system oversight.
- **Light/Dark Mode**: Fully responsive UI built with glassmorphism and modern aesthetics.
- **Toast Notifications**: Non-intrusive alerts for seamless UX.

## Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla), Bootstrap 5.
- **Backend**: PHP 8 (Custom MVC Architecture).
- **Database**: SQLite / MySQL (via PDO).

## Installation
1. Clone the repository.
2. Serve the application via PHP's built-in server:
   ```bash
   php -S localhost:8000 index.php
   ```
3. Navigate to `http://localhost:8000` in your browser.
